3. 添加修改文件
   在当前根目录下创建文件并添加内容
   1) 添加一个文件  git add 文件名
   $ git add index.html
   添加全部文件  
   $ git add .

4. 提交已经添加标记的文件到本地仓库
  $ git commit -m "备注信息"

5. 第一次本地代码提交到远程
   1) 在github上有一个远程仓库
   2) 在本地运行git romote add命令

    $ git remote add origin 远程仓库的地址
   3) 将本地代码push到远程仓库
    $ git push -u  远程仓库分支名称  本地仓库分支名称
    $  git push -u origin master
